# FlightFinder

## Project Description
FlightFinder is a project designed to analyze and document the complete process of flight search and booking systems. This repository contains planning, requirement analysis, architecture, and testing documents.

## Documents Included
- Problem Statement
- Brainstorming & Idea Generation
- Empathy Map
- Data Flow Diagrams
- Solution Architecture
- Solution Requirements
- Technology Stack
- Project Planning Template
- User Acceptance Testing

## Tools & Technologies
- Documentation: MS Word
- Planning & Analysis Techniques
- System Design Concepts

## Purpose
This project was developed as part of an academic / learning exercise to understand software development life cycle (SDLC) and system design.
